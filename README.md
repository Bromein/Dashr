## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm/yarn

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https:://github.com/bromein/repo.git
```

2. Install yarn packages

```sh
yarn install
```
